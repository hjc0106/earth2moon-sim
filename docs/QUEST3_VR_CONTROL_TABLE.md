# Quest3 VR 手柄控制表

Quest3 连接 CloudXR 后，先用键盘 `F3` 或 `F4` 选择机器人，再把双手柄放到希望对应机器人双末端的起始姿态，同时按一次左手柄 `Y` 和右手柄 `B` 完成标定。

## 通用操作

| 输入 | 功能 | 说明 |
| --- | --- | --- |
| 左手柄 6DoF 位姿 | 控制左臂末端位置和姿态 | 使用标定后的相对位姿。 |
| 右手柄 6DoF 位姿 | 控制右臂末端位置和姿态 | 使用标定后的相对位姿。 |
| 左 `Trigger` | 按住闭合左夹爪，松开张开 | 不冻结左臂末端。 |
| 右 `Trigger` | 按住闭合右夹爪，松开张开 | 不冻结右臂末端。 |
| 左右 `Squeeze` | 暂未绑定 | 正常握持不会影响机械臂。 |
| 左 `Y` + 右 `B` | 标定双手柄与双末端零位 | 切换机器人或按 `F1` 后需要重新标定。 |
| 头显旋转 | 改变机器人头部观察方向 | 头显平移受限，视点保持绑定到当前机器人头部相机。 |

## Ranger Arm

| 输入 | 功能 |
| --- | --- |
| 左摇杆上 / 下 | 小车前进 / 后退 |
| 左摇杆左 / 右 | 小车左转 / 右转 |
| 右摇杆 | 不控制 Ranger 底盘 |
| 左右手柄位置和旋转 | 分别控制左右机械臂末端 |
| 左右 `Trigger` | 分别控制左右夹爪闭合 / 张开 |

Ranger 使用 `F3` 进入头部视角和 VR 控制。当前头部视角固定挂在小车本体前上方并朝车体前方略向下看；底盘不提供横向平移，只提供沿视角前方的前后行驶和转向。

## R1 Pro

| 输入 | 功能 |
| --- | --- |
| 左摇杆上 / 下 | 底盘前进 / 后退 |
| 左摇杆左 / 右 | 底盘左移 / 右移 |
| 右摇杆左 / 右 | 底盘左转 / 右转 |
| 左手柄 `X / Y` | `torso_joint1` 负向 / 正向 |
| 右手柄 `A / B` | `torso_joint2` 负向 / 正向 |
| 左右手柄位置和旋转 | 分别控制左右机械臂末端 |
| 左右 `Trigger` | 分别控制左右夹爪闭合 / 张开 |

R1 Pro 使用 `F4` 进入头部视角和 VR 控制。腰部后两个关节当前只保留键盘控制。

## 辅助键盘

| 按键 | VR 场景中的作用 |
| --- | --- |
| `F1` | 返回启动时全场景视角并清除 VR 标定 |
| `F3` | 选择 Ranger、切头部视角并等待重新标定 |
| `F4` | 选择 R1 Pro、切头部视角并等待重新标定 |
| `方向键` | 微调当前脚本相机的偏航和俯仰 |
