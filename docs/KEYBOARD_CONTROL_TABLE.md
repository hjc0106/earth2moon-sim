# 键盘控制表

启动后先点击 Isaac Sim viewport，使键盘焦点进入仿真窗口。场景默认保持启动时的全局视角，但控制器默认优先选择可用的 `r1pro`。

## 模式与视角

| 按键 | 功能 | 说明 |
| --- | --- | --- |
| `F1` | 返回启动时的全场景视角 | 不切换机器人控制器；会清除当前 Quest3 标定，重新进入机器人视角后需要再次标定。 |
| `1` / 小键盘 `1` | 选择 `ranger_arm` 控制器 | 只切控制对象，不自动切相机。 |
| `2` / 小键盘 `2` | 选择 `r1pro` 控制器 | 只切控制对象，不自动切相机。 |
| `F3` | 选择 `ranger_arm` 并切到头部视角 | 同时清除旧 VR 标定。 |
| `F4` | 选择 `r1pro` 并切到头部视角 | 同时清除旧 VR 标定。 |
| `F6` | 切到当前机器人的 `head_top` 相机 | 当前机器人由 `1/2` 或 `F3/F4` 决定。 |
| `F9` | 切到 `cf2x` 跟随相机 | 不等于接管无人机控制。 |
| `F10` | 切到 `cf2x_01` 跟随相机 | 不等于接管无人机控制。 |
| `方向键 ←/→` | 当前跟随相机左转/右转 | 仅对脚本创建的机器人或无人机相机有效。 |
| `方向键 ↑/↓` | 当前跟随相机抬头/低头 | 仅对脚本创建的机器人或无人机相机有效。 |
| `TAB` | 切换左臂、右臂、双臂 | 对当前激活机器人生效。 |
| `ESC` | 退出 | 仅启动时启用 `--quit-on-esc` 才生效。 |

## Ranger Arm

| 按键 | 功能 |
| --- | --- |
| `W / S` | 底盘前进 / 后退 |
| `A / D` | 底盘左转 / 右转 |
| `I / K` | 当前机械臂关节 1 正向 / 负向 |
| `L / J` | 当前机械臂关节 2 正向 / 负向 |
| `U / O` | 当前机械臂关节 3 正向 / 负向 |
| `T / G` | 当前机械臂关节 4 正向 / 负向 |
| `F / H` | 当前机械臂关节 5 正向 / 负向 |
| `R / Y` | 当前机械臂关节 6 正向 / 负向 |
| `7 / 8` | 当前机械臂关节 7 正向 / 负向 |
| `M / N` | 夹爪打开 / 闭合 |

## R1 Pro

| 按键 | 功能 |
| --- | --- |
| `W / S` | 底盘前进 / 后退 |
| `A / D` | 底盘左移 / 右移 |
| `Q / E` | 底盘左转 / 右转 |
| `I / K` | 当前末端目标前移 / 后移 |
| `J / L` | 当前末端目标左移 / 右移 |
| `U / O` | 当前末端目标上移 / 下移 |
| `T / G` | 当前末端 Roll 正向 / 负向 |
| `F / H` | 当前末端 Pitch 正向 / 负向 |
| `R / Y` | 当前末端 Yaw 正向 / 负向 |
| `7 / 8` | 当前臂第 7 关节正向 / 负向 |
| `M / N` | 夹爪打开 / 闭合 |
| `Z / X` | `torso_joint1` 正向 / 负向 |
| `C / V` | `torso_joint2` 正向 / 负向 |
| `B / P` | `torso_joint3` 正向 / 负向 |
| `5 / 6` | `torso_joint4` 正向 / 负向 |

## 无人机

| 按键 | 功能 |
| --- | --- |
| `3` | 接管 `cf2x` |
| `4` | 接管 `cf2x_01` |
| `F2` | 退出无人机接管，回到当前地面机器人控制 |
| `W / S` | 无人机前进 / 后退 |
| `A / D` | 无人机左移 / 右移 |
| `Q / E` | 无人机下降 / 上升 |
| `J / L` | 无人机向左 / 向右偏航 |

`3/4` 当前用于无人机接管，不再用于机械臂第 7 关节；机械臂第 7 关节使用 `7/8`。
